# Remaining Helper Issues for Civilization VI

This file tracks remaining helper function issues for Civilization VI.

## Status

No remaining helper issues. All helper functions implemented and working correctly!

The following helper functions are implemented in `frontend/modules/shared/gameLogic/civ_6/civ6Logic.js`:
- `has`: Check if player has an item
- `count`: Count how many of an item the player has
- `location_item_name`: Get the item placed at a specific location
- `has_non_progressive_items`: Check if player has all required non-progressive items for an era
- `has_progressive_items`: Check if player has all required progressive items (with counts) for an era
